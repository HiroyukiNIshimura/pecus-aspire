# Tailwind CSS 任意値（Arbitrary Values）禁止ルール

## AI エージェント向け要約（必読）

- **ルール**: `z-[10]`, `w-[200px]` などの角括弧 `[...]` 構文は**絶対禁止**。
- **対処法**:
  - 定義済みクラス (`z-10`, `w-64`) を使用する。
  - 足りない場合は `tailwind.config.js` を拡張するのではなく、近似のクラスで妥協するか、デザインを見直す（FlyonUI の標準に従う）。
- **例外**: なし。

## 概要

Tailwind CSSの角括弧 `[...]` を使った任意値構文は**禁止**。

## 禁止対象

```
❌ z-[10], z-[100], z-[9999]
❌ w-[200px], h-[50vh], max-h-[90vh]
❌ text-[14px], text-[#ff0000]
❌ p-[20px], m-[2rem], gap-[10px]
❌ top-[50%], left-[calc(100%-20px)]
```

## 代替方法

```
✅ z-10, z-20, z-50
✅ w-full, h-screen, max-h-screen
✅ text-sm, text-base, text-lg
✅ p-4, m-8, gap-2
```

必要な値は `tailwind.config.js` の `extend` で定義する。

## 理由

1. デザインシステムの一貫性維持
2. 予測可能なスタイル管理
3. コードレビューの容易化
4. FlyonUI との整合性確保

## 検出パターン（正規表現）

```regex
\b\w+-\[.+?\]
```

## 注意

Biome には Tailwind 任意値を検出するルールがないため、AIエージェントおよびコードレビュー時に手動で確認すること。
