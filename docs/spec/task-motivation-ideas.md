# Pecus Aspire — タスク達成モチベーション向上アイディア集

最終更新: 2026-01-11

## 概要

Pecus Aspire（Coati）は、業務アプリでありながら「遊び心」を取り入れ、ユーザーのモチベーションを高める機能を検討しています。以下は、グローバル公開・業務利用でも違和感なく使える“楽しい体験”のアイディア集です。

---

### 1. タスク達成時のミニアニメーション
- 担当者がタスク完了時（もしくは進捗100%）、画面端で小さな動物（コアティ等）が祝福するアニメーション。
- 例: コアティが「やったね！」とジャンプ → 2秒で消える。
- Tailwind/FlyonUIで実装可能。邪魔にならないサイズ。


### 2. 称号システム
- 一定条件（連続達成日数、緊急度の高いタスク完了等）で称号を付与。
- 例: 「朝型コアティ」「スプリントマスター」など。
- プロフィールやタスク一覧に小さく表示。
- **生成AIによる称号自動生成:** ユーザーのアクティビティや達成内容をAIに渡し、ユニークな称号を自動生成する仕組みも導入可能。例: 「今週の閃き王」「リーダーシップ・コアティ」など、個人の特徴や成果に応じた称号をAIが提案。
- **公開範囲:** デフォルトは「個人のみ閲覧」。オプションで「組織内公開」も選択可能（ランキングや称号一覧で表示）。管理者が全体公開を強制する設定も可。

### 3. コアティコレクション
- タスク達成で“コアティ”のイラストやスタンプがランダムで集まる。
- コレクション画面で一覧表示。
- 収集率が上がると特別なアニメーション。
- **公開範囲:** デフォルトは「個人のみ閲覧」。オプションで「組織内公開」も選択可能（コレクション率ランキングや他ユーザーのコレクション閲覧など）。

## 称号システムとコアティコレクションの違い・補完関係

- **称号システム**: ユーザーの行動や成果に応じて「肩書き」や「ラベル」が付与される。例: リーダーシップ、継続力、スピードなど“抽象的な評価”や“個性”を表現。
- **コアティコレクション**: タスク達成ごとに“具体的なアイテム（イラスト・スタンプ）”が集まる。例: ガチャ的な楽しみ、収集率、コンプリート欲を刺激。
- **補完関係**: 称号は「実績・個性の見える化」、コレクションは「達成の積み重ね・遊び心」。両方あることで「評価される楽しさ」と「集める楽しさ」が両立できる。

### 4. ランダム応援メッセージ
- タスク完了時、AI生成の一言応援メッセージを表示。（DM）
- 例: 「今日も一歩前進！」「コアティも応援してるよ！」

## 企業ごとの運用例

- **Amazon等のグローバル企業**
	- 称号やコレクション、達成状況を「オフィスのモニター」や「社内ポータル」で公開し、全体の士気や一体感を高める運用が多い。
	- 競争・称賛・透明性を重視する文化にマッチ。

- **プライバシー重視の企業・チーム**
	- 称号やコレクションは「自分だけが見られる」設定が好まれる。
	- 比較やプレッシャーを避けたい場合に有効。

このように、組織文化や方針に応じて公開範囲のデフォルトや選択肢を柔軟に設計することが重要。
