# 管理画面「組織設定」利用ガイド（エンドユーザー向け）

## この画面でできること
- 管理者メニュー → 設定 → 組織設定 で開きます。
- 組織全体のメール送信元、生成AI接続、タスク/チャット/ダッシュボード/ワークスペースの初期動作をまとめて決められます。
- 保存するとすぐに全ユーザーへ反映されます。入力に問題がある場合はエラーメッセージが表示されます。

## 画面構成と項目

### 1. 週間レポート
- **週間レポート配信曜日**: 日〜土のボタンから1つ選びます。選んだ曜日がハイライトされます。

### 2. メール送信元
- **メール配信元アドレス**: 必須。通知メールの送り主アドレスになります。
- **メール配信元名**: 必須。受信者に表示される名前です。

### 3. 生成AI設定
- **生成APIベンダー**: 必須。`未設定 / OpenAI / Anthropic / Google Gemini / DeepSeek` から選びます。
- **生成APIキー**: ベンダーを選んだら入力してください。ベンダーが「未設定」のときは入力不要・保存もされません。
- **生成AIモデル**:
  - APIキーを入力してフォーカスを外すと利用可能なモデルが自動で出ます。
  - 今設定中のモデルが一覧にない場合は警告が表示されます。選び直してください。
  - 右側の再読み込みボタンでいつでも一覧を取り直せます。
- **プラン**: 必須。`Free / Standard / Enterprise` から選びます。

### 4. タスク設定
- **タスク超過閾値（日）**: 必須。0〜365 の整数で設定。+/- ボタンで微調整できます。
- **ヘルプコメント通知先**: `組織全体 / ワークスペースユーザー` から選びます。
- **タスク作成時に見積もりを必須とする**: ON にするとタスク作成時に見積もり入力を求めます。
- **先行タスク完了を強制する**: ON にすると先行タスクが終わるまで後続タスクを操作できなくなります。

### 5. ダッシュボード設定
- **ヘルプコメント表示件数**: スライダーで 5〜20 件を指定。「ヘルプを求めているタスク」の最大表示数です。

### 6. チャット設定
- **グループチャットのスコープ**: `ワークスペース単位 / 組織単位` を選びます。

### 7. ワークスペース設定
- **デフォルトのワークスペースモード**: `通常 / ドキュメント` を選びます。新しいワークスペース作成時の初期モードになります。

## 操作
- **保存**: 入力内容を保存します。成功するとトーストでお知らせ。ほかの人が同時に更新していた場合は最新内容で自動的に上書きされ、警告が出ます。
- **リセット**: 入力途中の内容を破棄し、画面を開いたときの状態に戻します。

## バリデーション/エラー表示
- 必須漏れや範囲外の値はフィールド下にエラーが出ます。全体のエラー件数もトーストで知らせます。
- モデル取得に失敗した場合は「モデル一覧の取得に失敗しました」などのメッセージが出て、一覧がいったん空になります。

## 利用時のヒント
- ベンダーを変えるとAPIキーとモデルは一度クリアされます。再入力してください。
- モデル一覧が出ないときは、APIキーを入力して別の場所をクリックするか、再読み込みボタンを押してください。
- ほかの管理者が同時に保存した場合は、その人の最新内容が反映されたうえで続きから編集できます。
