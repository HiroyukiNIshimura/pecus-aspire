# 管理画面「組織編集」利用ガイド（エンドユーザー向け）

## この画面でできること
- パス: 管理者メニュー → 組織 → 組織編集
- 組織の名前や代表者、連絡先、概要を更新できます。
- 保存するとすぐに反映されます。入力に誤りがある場合はフィールド下にエラーが表示されます。

## 画面構成

### 基本情報（閲覧のみ）
- **組織ID / 組織コード**: システムで発行された識別子。編集不可。
- **所属ユーザー数**: 現在の所属人数。
- **作成日時**: 組織が作成された日時。

### 基本情報編集（入力エリア）
- **組織名**: 必須。組織の正式名称。
- **代表者名**: 任意。代表者の氏名。
- **メールアドレス**: 任意。連絡用メール。
- **電話番号**: 任意。連絡先電話番号。
- **説明**: 任意。組織の補足説明やメモ。

### 詳細情報（閲覧のみ）
- **更新日時**: 直近で組織情報が更新された日時。

## 操作
- **更新**: 入力内容を保存します。処理中はスピナー表示、完了すると通知が出ます。
- **キャンセル**: 変更を破棄し、一覧ページ（/admin）へ戻ります。
- **一覧に戻る**: 画面上部のボタンでも一覧へ戻れます。

## バリデーション/エラー表示
- 必須漏れや形式エラーがあるフィールドには赤枠とエラーメッセージが表示されます。
- サーバー側エラーがある場合は画面上部にエラーアラートが表示されます。

## 利用時のヒント
- 入力途中でもキャンセルで一覧へ戻れます（内容は保存されません）。
- 代表者名・連絡先・説明は任意ですが、問い合わせ対応を想定する場合はできるだけ入力してください。
- 他の管理者と同時に更新した場合は、再読み込み後に最新の状態を確認してください。
