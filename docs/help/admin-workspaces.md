# 管理画面「ワークスペース管理」利用ガイド（エンドユーザー向け）

## この画面でできること
- 管理者メニューからワークスペース一覧を確認・検索できます。
- 名前、ジャンル、ステータスで条件を絞り込み、ページ切り替えができます。
- ワークスペースの編集画面へ遷移したり、削除を実行できます。
- 登録数やメンバー数などの集計サマリーを確認できます。

## 画面構成

### ヘッダー / サイドバー
- 共通の管理ヘッダーとサイドバーで他の管理ページへ移動できます。

### フィルター
- 開閉式のフィルターセクションで検索条件を設定します。
- ワークスペース名: 前方一致で検索します。任意入力で100文字以内（Zod: `workspaceNameFilterSchema`）。
- ジャンル: プルダウンで選択。未選択の場合はすべてのジャンルが対象です。
- ステータス: アクティブ / 非アクティブ / すべて を切り替え。初期値はアクティブのみ。
- リセット: 条件を初期状態（名前空、ジャンル未選択、ステータス=アクティブ）に戻し、一覧を再取得します。
- 検索: 現在の条件で一覧を再取得します。

### ワークスペース一覧
- 列: コード / ワークスペース名 / ジャンル / 参加者数 / ステータス / 作成日 / 操作。
- ステータスはアクティブ / 非アクティブのバッジで表示されます。
- 操作:
  - 編集: `/admin/workspaces/edit/<ワークスペースID>` へ遷移します。
  - 削除: 確認モーダルを表示し、承認後に削除します。

### ページネーション
- 総件数と現在のページ番号を確認しながらページを切り替えできます。

### サマリー
- 集計カードで統計を表示します。
  - 総ワークスペース数
  - アクティブ / 非アクティブの件数と比率
  - 総メンバー数（重複なし）
  - 平均メンバー数（ワークスペースあたり）
  - 最近作成されたワークスペース数（過去30日）

### 削除確認モーダル
- 削除ボタン押下で対象ワークスペースの確認モーダルが表示されます。
- 承認すると削除を実行し、一覧が再読み込みされます。

## 操作の流れ
- 必要に応じてフィルターを開き、条件を設定します。
- 検索を押して一覧を更新します。
- 編集または削除を行うワークスペースを選択します。
- 件数が多い場合はページを切り替えて目的の行を探します。

## バリデーション / 入力ルール
- ワークスペース名: 任意入力。100文字以内で入力してください。

## 注意事項
- 不明な項目や未定義の内容は ○○○○ のままです。
