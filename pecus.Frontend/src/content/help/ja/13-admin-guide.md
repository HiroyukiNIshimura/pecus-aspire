# 管理者向け機能ガイド

## 組織管理者画面

特定の組織を管理するための画面です。組織の管理者が利用します。

### アクセス方法
サイドメニューまたはヘッダーの管理者メニューからアクセスします。

### 主な機能

| メニュー | 機能概要 |
|---|---|
| **組織** | 組織の基本情報（名称、コード、代表者情報など）を確認できます。 |
| **設定** | 組織全体に関わる設定を行います。 |
| **ワークスペース** | 組織内のワークスペースを作成・編集・削除します。プロジェクトやチームごとにワークスペースを分割して管理できます。 |
| **ユーザー** | 組織に所属するメンバーを管理します。招待、権限変更、削除などが可能です。 |
| **スキル** | 組織内で利用するスキル（技術スタックや資格など）を定義します。ユーザーのスキルマッチング機能などで利用されます。 |
| **タグ** | アイテムやドキュメントに付与するタグを管理します。 |
