# 管理者向け機能ガイド

## 組織管理者画面

特定の組織を管理するための画面です。組織の管理者が利用します。
組織内で管理者権限を持つユーザーのみが利用できます。

### アクセス方法
サイドメニューまたはヘッダーの管理者メニューからアクセスします。

![画面](/help/images/1000021.webp)

### 主な機能
| メニュー | 機能概要 |
|---|---|
| **組織** | 組織の基本情報（名称、コード、代表者情報など）を確認できます。 |
| **設定** | 生成AIの設定他、組織全体に関わる設定を行います。 |
| **ワークスペース** | 組織内のワークスペースを作成・編集・削除します。プロジェクトやチームごとにワークスペースを分割して管理できます。 |
| **ユーザー** | 組織に所属するメンバーを管理します。招待、権限変更、削除などが可能です。 |
| **スキル** | 組織内で利用するスキル（技術スタックや資格など）を定義します。ユーザーのスキルマッチング機能などで利用されます。 |
| **タグ** | アイテムやドキュメントに付与するタグを管理します。 |

### AIを利用するには
AIを利用するには組織設定で利用する生成AIのAPI情報を設定する必要があります。
生成AIベンダーを選択し、組織で取得したAPIキーを入力、利用可能なモデルを選択する必要があります。
![画面](/help/images/1000030.webp)
> 画像や音声生成AIモデルは除外していますが、新しいバージョンのモデルが表示されてしまう場合もあります。その場合、文章生成AIモデル以外を選択しても正しく機能しないので注意してください。
