# エディタ機能

Coatiでは、Notion ライクなリッチテキストエディタを採用しており、直感的な操作でドキュメントやアイテムの説明を作成できます。

## 基本操作

![画面](/help/images/1000020.webp)

### スラッシュコマンド
段落の先頭で `/` を入力すると、メニューが表示され、様々な要素を挿入できます。

**利用可能なコマンド:**
- **基本ブロック**: 見出し (H1-H3)、段落、引用、コードブロック、区切り線、改ページ
- **リスト**: 番号付きリスト、箇条書きリスト、チェックリスト
- **メディア**: 画像、動画 (YouTube)、ツイート (X)、Figma
- **レイアウト**: テーブル、カラムレイアウト、折りたたみ (Collapsible)
- **その他**: 日付、数式、AI アシスタント (有効な場合)

### Markdown ショートカット
Markdown 記法を使用して、素早くフォーマットを適用できます。

| 機能 | 入力 |
|---|---|
| 見出し 1 | `# ` + Space |
| 見出し 2 | `## ` + Space |
| 見出し 3 | `### ` + Space |
| 箇条書きリスト | `- ` または `* ` + Space |
| 番号付きリスト | `1. ` + Space |
| チェックリスト | `[] ` + Space |
| 引用 | `> ` + Space |
| コードブロック | ` ``` ` |
| 区切り線 | `---` または `***` |
| 太字 | `**text**` |
| 斜体 | `*text*` |
| 取り消し線 | `~text~` |
| インラインコード | `` `text` `` |
| リンク | `[text](url)` |
| 数式 | `$equation$` |

## 高度な機能

### テーブル
スラッシュコマンドから `Table` を選択するか、Markdown 記法 (`| col1 | col2 |`) でテーブルを作成できます。
セル内で右クリックすると、行や列の追加・削除などの操作が可能です。

> Excelなどのスプレッドシートで作成表のクリップボードから貼り付けもテーブルに自動で変換されます。

### 画像とメディア
- **画像**: スラッシュコマンドから `Image` を選択するか、画像をエディタにドラッグ＆ドロップしてアップロードできます。
- **埋め込み**: YouTube、X (Twitter)、Figma の URL を貼り付けるか、スラッシュコマンドから選択して埋め込むことができます。

### 数式
LaTeX 形式の数式をサポートしています。スラッシュコマンドから `Equation` を選択するか、`$` で囲んでインライン数式を入力できます。

### Markdown
クリップボードからMarkdownを貼り付けることで、エディタ内にMarkdownでのスタイルを再現できます。ただし、GFMなど全ての記法には対応していません。

### AI アシスタント
AI 機能が有効な場合、`/` メニューから `AI Assistant` を呼び出し、文章の生成や修正を依頼できます。

### ハッシュタグ
`#` に続けてアイテムコードを入力すると、ハッシュタグとして認識されます。ただし、ワークスペースを横断しての利用には対応していません。

### その他のオートリンク
- ワークスペースコード#アイテムコード（例：Oc76lxrXKmc0ifCo#1）
- ワークスペースコード#アイテムコードTタスクシーケンス（例：Oc76lxrXKmc0ifCo#1T1）

この記法でもオートリンクを作成します。ワークスペースを横断したリンクを作成したい場合はこちらを利用してください。

> ただし、ワークスペースへのアクセス権限がないユーザーはアクセスすることはできません。
