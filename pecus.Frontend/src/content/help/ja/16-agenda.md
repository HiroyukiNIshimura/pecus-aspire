# イベント機能

イベント機能を使って、予定の作成・管理・参加状況の回答ができます。チームメンバーとの予定調整がスムーズになります。

## イベント一覧

サイドメニューの「イベント」をクリックすると、直近の予定がタイムライン形式で表示されます。
![画面](/help/images/1000024.webp)

### タイムライン表示
予定は時系列で整理されます。

| セクション | 内容 |
|-----------|------|
| 今日 | 本日の予定 |
| 明日 | 明日の予定 |
| 今週 | 今週中の予定 |
| 来週 | 来週の予定 |
| 来週以降 | さらに先の予定 |

### 表示の見方
| 表示 | 意味 |
|-----|------|
| ハイライトの背景 | 未回答の予定（要対応） |
| 終日 バッジ | 終日イベント |
| 毎週 などのバッジ | 繰り返しイベント |
| 取り消し線 | 中止されたイベント |

タイムライン上で参加、仮、不参加の設定も行えます。

## 新しいイベントを作成する

画面右上の「+ 新規作成」ボタンから作成できます。

### 基本情報

![画面](/help/images/1000025.webp)

#### 必須項目
| 項目 | 説明 |
|------|------|
| タイトル | 予定の名前（最大200文字） |
| 開始 | 開始日時 |
| 終了 | 終了日時 |

#### 任意項目
| 項目 | 説明 |
|------|------|
| 終日 | チェックで時刻指定なしの1日イベントに |
| 場所 | 会議室名、住所など |
| URL | Zoom/Teamsなどの会議リンク |
| 詳細 | イベントの説明（Markdown対応） |

🫆 アイテムエディタと同じくオートリンクが利用できます
- ワークスペースコード#アイテムコード（例：Oc76lxrXKmc0ifCo#1）
- ワークスペースコード#アイテムコードTタスクシーケンス（例：Oc76lxrXKmc0ifCo#1T1）

### 繰り返し設定

定例会議などの繰り返しイベントを設定できます。

| 種類 | 説明 |
|------|------|
| 繰り返しなし | 単発イベント |
| 毎日 | 毎日繰り返し |
| 毎週 | 毎週同じ曜日 |
| 隔週 | 2週間ごと |
| 毎月（日付） | 毎月同じ日（例：毎月15日） |
| 毎月（曜日） | 毎月同じ週・曜日（例：第2火曜日） |
| 毎年 | 毎年同じ日 |

**終了条件**
- 終了日を指定：特定の日付まで繰り返し
- 回数を指定：決まった回数まで繰り返し（例：12回）

### リマインダー

予定の前に通知を受け取れます。複数選択できます。

- 1日前
- 1時間前
- 30分前
- 15分前
- 開始時

### 参加者の追加

**クイック追加**
- 「全員追加」：組織の全メンバーを一括追加
- ワークスペースから追加：特定のワークスペースメンバーを一括追加
- 参加者は100名を上限としています。

**個別検索**
検索ボックスに名前やメールアドレスを入力して検索・追加できます。

**参加者の設定**
- ×ボタン：参加者を削除
- 任意参加：必須ではない参加者として設定可能

### 通知設定

「参加者にメール通知を送信する」をオンにすると、参加者に招待メールが届きます。

## 参加状況に回答する

招待されたイベントには、参加状況を回答しましょう。

### クイック回答（一覧画面から）

各予定の右側にあるボタンで素早く回答できます。

| ボタン | 意味 |
|--------|------|
| ✅ 参加 | 参加します |
| ❓ 仮 | 未定・検討中 |
| ✖️ 不参加 | 参加しません |

### 詳細画面から回答

1. 予定のタイトルをクリック
2. 「参加状況を変更」セクションで回答

### 繰り返しイベントの回答

繰り返しイベントでは、特定の回だけ回答を変えることもできます。

- **この回のみ**：クリックした日程だけ変更
- **この回以降すべて**：選んだ日以降のすべての回を一括変更

## イベントを編集・中止する

### 編集する

1. イベントの詳細画面を開く
2. 「編集」ボタンをクリック
3. 内容を変更して「保存」

**繰り返しイベントの編集**

繰り返しイベントを編集するときは、範囲を選べます。

| 選択肢 | 説明 |
|--------|------|
| この回のみ | 選んだ日程だけ変更 |
| この回以降すべて | 選んだ日以降を変更（新しいシリーズが作成されます） |
| すべての予定 | シリーズ全体を変更 |

### 中止する

1. イベントの詳細画面を開く
2. 「中止」または「シリーズを中止」ボタンをクリック
3. 中止理由を入力（任意）
4. 「中止する」で確定

中止されたイベントは一覧に取り消し線で表示され、参加者に通知されます。

## 繰り返しイベントの使い方

### 特定の回だけ変更・中止したい

繰り返しイベントの一部だけ変更することができます。

1. 変更したい日程をクリックして詳細画面へ
2. 「編集」または「この回を中止」を選択
3. 「この回のみ」を選んで変更

変更された回は詳細画面の「変更・中止された回」セクションに記録されます。

## よくある質問

### 参加状況を間違えて回答してしまった
いつでも変更できます。同じボタンをもう一度押すか、別の回答を選んでください。

### 繰り返しイベントの途中から参加状況を変えたい
詳細画面の「この回以降の一括設定」を開いて、まとめて変更できます。

### 予定の詳細にリンクや箇条書きを入れたい
詳細欄はMarkdownに対応しています。

```
- 箇条書き
- **太字**
- [リンク](https://example.com)
```

### 通知が届かない
以下を確認してください。
- イベント作成時に「参加者にメール通知を送信する」がオンになっているか
- 迷惑メールフォルダに入っていないか
- リマインダーが設定されているか

### 中止したイベントを復活させたい
現在、中止の取り消しはできません。新しいイベントを作成してください。

## 💡 ヒント

- **未回答の予定は行ハイライト**：見逃さずに回答しましょう
- **クイックボタンで素早く回答**：一覧画面から1クリックで完了
- **繰り返し設定を活用**：定例会議は一度の設定でOK
- **リマインダーを複数設定**：重要な予定は1日前＆1時間前など
- **ワークスペースから一括追加**：メンバー追加が楽になります
